## NDVI Vegetation Density Map of Hamburg (Summer 2025)

This project presents an NDVI-based vegetation density map of Hamburg using Sentinel-2 imagery processed in Google Earth Engine and visualized in QGIS.

## Tools Used
- Google Earth Engine
- QGIS
- Sentinel-2 Satellite Imagery

## Output
Final NDVI vegetation density map showing vegetation distribution across Hamburg.

## Po River Delta NDTI Analysis

This project uses Sentinel-2 imagery in Google Earth Engine to calculate the Normalized Difference Turbidity Index (NDTI) for the Po River Delta region.

The workflow includes:
- NDWI-based water masking
- NDTI calculation
- Turbidity visualization
- Raster export for further GIS analysis

### Tools Used
- Google Earth Engine
- Sentinel-2 SR Harmonized
- QGIS
